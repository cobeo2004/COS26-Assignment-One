CSS Heading Typing Animation

This enhancement uses advanced CSS such as keyframes and animations.

The animation is done by placing the page heading into a container, then gradually increasing the width of the container to make it look like the text is being typed out. For the blinking cursor animation, the right border of the container alternates between solid black and transparent.

Due to each heading having different lengths, the CSS must be adapted individually to fit each heading.

The animation is done using only HTML and CSS, and the keyframes and animations go beyond what was taught in the lectures. Overall, the aim of this enhancement was to make the website more dynamic and enjoyable to use, therefore increasing the quality of the website and making it more engaging to users.
